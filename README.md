# 🎵 Studio Multimídia

https://github.com/jwalmirdejesus-netizen/studio_multimidia/blob/main/Screenshot_20260523-105916.jpg

Um estúdio criativo completo que roda no seu navegador. Feito para celular primeiro, com design dark, rápido e sem precisar instalar nada.

> Corte áudio, junte faixas, converta vídeo para MP3, grave voz, edite fotos e muito mais — tudo offline.

## ✨ Features

### Edição de Áudio
- ✂️ **Cortar** — cortes precisos por tempo
- 🧩 **Juntar** — une vários áudios em um só
- 🎚️ **Mixar** — sobreposição de faixas
- 🪓 **Dividir** — separa em partes
- 📥 **Inserir Áudio** — adiciona trechos
- 🏷️ **Tags ID3** — edita metadados

### Processamento
- 🎬➡️🎵 **Vídeo para MP3**
- 🔊➡️🎬 **Áudio para Vídeo**
- 🔄 **Converter Formato**
- 🗜️ **Compressor** de áudio
- 🎙️ **Gravar** com microfone

### Criativo
- 🖼️ **Editor de Fotos**
- 🎞️ **Editor de Vídeo**
- 🎛️ **Equalizador**

### Extra
- 📂 **Histórico local** com IndexedDB — seus exports ficam salvos no aparelho
- 📱 **Interface mobile** com bottom navigation
- 🌌 Design com gradientes e glassmorphism
- ⚡ 100% client-side — nada vai para servidor

## 🛠️ Tecnologias
- HTML5 + CSS3 (variáveis, grid, backdrop-filter)
- JavaScript Vanilla (sem frameworks)
- IndexedDB para armazenamento
- Iframe sandbox para ferramentas modulares
- PWA-ready
